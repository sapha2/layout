Flexbox - ein zweidimensionales Kastensystem

Die einfachste Art, um den Inhalt einer Seite eingiermaßen zu formatieren, ist die Flexbox.
Hierbei wird die Möglichkeit genutzt, verschiedene Objekte ineinander zu verschachteln. Der Fachbegriff ist
Nesting

1. Nesting

Besimmt schonmal passiert: du hast vergessen, einen Tag wieder zu schließen.
Eventuell wolltest du

nur mal kurz eine Überschrift schreiben. Schon war der Text schief.
Tatsächlich ist das die einfachste Form von "Nesting". Eigentlich spricht man erst davon, wenn ein Objekt einem andern bewusst untergeschoben wird. Ein Beispiel sind die beiden nächsten Divs

Ich bin eine einsame Box.
Ich bin das sogenannte parent element
Ich bin das erste child element
Ich bin das zweite child element

Sicher fällt dir direkt auf, was da passiert ist. Das parent Element ist also das, welches erst nach dem
child Element wieder geschlossen wird. Alle child Elemente übernehmen übrigens die Eigenschaften des parent Elements.
Man spricht auch - wie im echten Leben - von "erben" oder "inherit"

Definiert man nun ein "container" Div als parent weiterer Elemente, so kann man über
die "Anzeigen" Eigenschaft dem Browser mitteilen, dass er den container als Layoutsystem nutzen soll.

!

2. Flexbox

Und schon geht es los. Hast du dich schon gewundert, warum in diesem Dokument alle bisherigen Elemente in der Mitte der Seite dargestellt werden?
Schon komisch, oder? Suche nach der Ursache. Ein kleiner Tip: da es alle Elemente betrifft, muss die Eigenschaft für ein sehr wichtiges Element festgelegt worden sein...
